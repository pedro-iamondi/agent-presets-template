# Scratch

Use esta pasta para outputs temporários de agentes, rascunhos descartáveis, relatórios intermediários e experimentos locais.

Regras:

- Não tratar arquivos desta pasta como fonte de verdade.
- Não guardar secrets, tokens, dados sensíveis ou arquivos de cliente.
- Mover qualquer decisão reutilizável para documentação apropriada antes de finalizar a tarefa.
- Limpar conteúdo temporário quando ele não for mais útil.
